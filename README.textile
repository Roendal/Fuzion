h2. Installation

Add to your Gemfile:
<pre><code>
  gem 'fuzion_rpg'
</code></pre>
Then run:
<pre><code>
  bundle update
</code></pre>
Run install script:
<pre><code>
  rails g fuzion_rpg:install
</code></pre>
And don't forget to migrate you database:
<pre><code>
  rake db:migrate
</code></pre>
